Guide d'Utilisation – Secrétaire

1. Premiers Pas

Bienvenue dans le système de gestion Elite Robotique. Après connexion, vous avez accès aux modules suivants :

  • 📝 Inscription : Gérer les nouvelles demandes
  • 🧑‍🎓 Membres : Voir et modifier les profils
  • 💵 Paiements : Enregistrer et suivre les paiements
  • 📅 Présences : Marquer les présences hebdomadaires
  • 📊 Rapports : Générer des statistiques

2. Gérer les Inscriptions

  1. Allez dans Inscription → En attente
  2. Vérifiez les documents : pièce d’identité, photo, paiement
  3. Cliquez sur Valider ou Demander des modifications
  4. Le membre est ajouté automatiquement à la liste

3. Enregistrer un Paiement

  1. Allez dans Paiements → Enregistrer
  2. Sélectionnez le membre et le mois concerné
  3. Entrez le montant et le mode de paiement
  4. Cliquez sur Enregistrer

4. Marquer les Présences

  1. Allez dans Présences → Marquer
  2. Sélectionnez la date et le groupe
  3. Cocher les membres présents
  4. Cliquez sur Enregistrer

5. Générer des Rapports

Allez dans Rapports → Statistiques et sélectionnez :

  • Présences : Taux de présence par mois
  • Financier : Paiements reçus / en retard
  • Inscriptions : Évolution mensuelle

Cliquez sur Exporter en PDF ou Excel.
