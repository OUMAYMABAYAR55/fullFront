FAQ – Secrétaire

Accédez à Membres → Nouveau membre, remplissez le formulaire et cliquez sur Enregistrer.

Vérifiez que le paiement a été enregistré avec le bon mois et membre. Si le problème persiste, contactez le gerant.

Allez dans Membres → Liste, cliquez sur le membre, puis sur Imprimer la fiche.

Le système signale automatiquement les membres avec +3 absences. Allez dans Présences → Absences répétées pour les contacter.

Allez dans Config → Documents pour télécharger les modèles (fiche d'inscription, reçu, etc.).
